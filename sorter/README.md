1) Написать консольную программу, которая бы сортировала текст поданный ей на стандартный вход по алфавиту.
2) Программа должна игнорировать регистр при сортировке
3) Программа должна сортировать не по алфавиту, а по количеству символов в строке если ей на вход дать ключ -D sorter.length
4) Программа в качестве аргумента может получать число, тогда считать что порядковый номер слова в строке, по которому надо сортировать строки в алфавитном порядке
